Currency Transfer API

REST API на Spring Boot для конвертации валют с использованием открытого сервиса Frankfurter API.

Проект демонстрирует основы разработки REST-приложений на Java с использованием архитектуры Controller → Service, обработки HTTP-запросов и работы с внешним API.

Возможности
Конвертация суммы из одной валюты в другую.
Получение актуального курса валют через Frankfurter API.
REST API с использованием Spring Boot.
Простая и понятная архитектура проекта.

Используемые технологии
Java 17
Spring Boot
Spring Web
Maven
RestTemplate
Frankfurter API
